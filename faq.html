FAQ

What is Working in Widener?

It's an educational game. You assume the role of a shelver working in Harvard's Widener Library. You are assigned five books to shelve and asked to find their places in the stacks using their call numbers. The faster the better.

A multiplayer game?

Yes, you can play another person. Up to two players. (Many two player games can be played at one time.) You can also play in single player mode. If you go this route, you're racing against the clock.

How do I find an opponent?

Working in Widener will pair two players up. So, load the game, choose two player and wait. When a second player does the same you'll be racing against each other. If you're waiting a while, email a buddy and tell them to join you or play in single player mode.

Is this really Widener Library?

Yes, but we simplified Widener. We sliced four floors out of Widener and reduced the number of shelves on each level. The floor plan is close to what you'll find in the physical world, the call number locations too.

Does each shelver get the same books?

Books are randomly chosen for each game. If you're playing in two player mode, you each get the same set of books, but we shuffle them in different orders.

Is this real book data?

Yes. Book data is loaded from the Harvard catalog using the LibraryCloud API

Is my time good?

Hopefully. See how you stack up against other players by looking at the Top Scores page.

This game is cool. How did you build it?

It's mostly JavaScript leveraging the Node.js and Socket.io technologies. A big thanks to those folks for building a sharing.

Can I get this for my library?

Yes! Our source code can be found on github. Drop us a note and let us know if you have something to share.
